# Plan You - Organizador de Tarefas

Um organizador de tarefas moderno e responsivo, com design minimalista e ícones do Google Material Design.

## Funcionalidades

- ✅ Gerenciamento de tarefas (adicionar, editar, excluir)
- 📊 Visualização de estatísticas (para começar, em andamento, concluídas, urgentes)
- 🏷️ Filtros por prioridade (rotina, urgente, importante, delegar, adiar)
- 📅 Visualização em agenda
- 📝 Visualização de notas
- 💾 Armazenamento local das tarefas
- 🔄 Integração com API pública para tarefas de exemplo

## Tecnologias Utilizadas

- **HTML5**: Estrutura da aplicação
- **CSS3**: Estilização com design minimalista e responsivo
- **JavaScript (ES6+)**: Logica da aplicação
- **Google Material Icons**: Ícones minimalistas
- **API JSONPlaceholder**: Tarefas de exemplo

## Como Usar

1. Clone este repositório ou baixe os arquivos
2. Abra o arquivo `index.html` em seu navegador
3. Ou execute um servidor local (ex: `python -m http.server` ou `npx serve`)
4. Comece a organizar suas tarefas!
